## Description

In this task, I performed Sentiment Analysis on customer reviews to identify whether a review is Positive or Negative. The dataset contains 20 customer reviews which are equally balanced with 10 positive and 10 negative reviews.

I used TF-IDF Vectorization to convert the text reviews into numerical values so that the machine learning model can understand them. TF-IDF stands for Term Frequency Inverse Document Frequency and it gives more importance to unique words in each review.

After vectorization, I trained a Logistic Regression model on the data. The model was tested and achieved 100 percent accuracy, meaning it correctly identified all positive and negative reviews.

I also tested the model with new unseen reviews and it correctly predicted the sentiment for each one. For example, a review saying amazing product was predicted as Positive and pathetic garbage waste of money was predicted as Negative.

This task helped me understand how Natural Language Processing works, how to convert text into numbers using TF-IDF, and how to build a text classification model using Logistic Regression.

Tools used in this task are Python, Scikit-learn, Pandas, TF-IDF Vectorizer, Logistic Regression, and Google Colab.

The output of this task is a Jupyter Notebook showing complete preprocessing, model training, evaluation, and sentiment predictions.
